  4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons
for using it)?

    
Answer:
- Unobtrusive Javascript: Uses the DOM to attach and execute all JavaScript functions.
- Idea behind 'Unobtrusive' is to keep script separate from HTML.
- Common 'obtrusive' example is <button onclick="someMethodCall();"> its not recommended approach.
- Recommended Unobtrusive approache is 
< script ...> 
 var buttonElement = document.getElementById("buttonId");
 buttonElement.onclick=someMethodCall; 
</script>
